# Tesla Stock Movement Prediction Based on News Sentiment and Machine Learning

This project studies Tesla next-day stock movement prediction using stock price data, technical indicators, market-related variables, financial news sentiment, and machine learning models.

## Project Structure

notebooks/          Jupyter notebooks
data/sample/        Sample input data location
data/raw/           Raw collected data
data/processed/     Processed datasets
logs/               Data collection logs
results/            Model results and output tables

## Main Methods

- Tesla stock data preprocessing
- Technical indicator generation
- Financial news sentiment analysis
- Feature fusion
- Time-series cross-validation
- Logistic Regression, Random Forest, XGBoost, and LightGBM comparison
- Feature-group ablation study
- Threshold sensitivity analysis

## Note

Large datasets and API keys are not included in this repository. Please place the required input files in the appropriate data folders before running the notebook.
