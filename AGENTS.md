# AGENTS.md

## Project Context

This is a final year project about Tesla stock price movement prediction using news sentiment analysis and machine learning.

The project is written mainly in Python and was originally developed in Kaggle Notebook.

## Main Goal

Help clean, organize, review, and explain the project code for academic final year project purposes.

## Important Rules

- Do not remove important experiment sections unless specifically requested.
- Do not expose or add any API keys, passwords, tokens, or private credentials.
- Keep the code readable and suitable for academic explanation.
- Prefer simple and explainable code over overly complex solutions.
- Keep the original notebook unless asked to refactor it.
- When modifying code, explain what was changed and why.
